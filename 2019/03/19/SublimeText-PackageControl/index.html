macOS Sublime Text 插件安装与删除

2019-03-19

Sublime Text 是收费软件，但可以无限期试用，作者很厚道地给用户们提供了免费无限制无限期的试用权，它只会偶尔提醒一下你木有购买，而且频率也很低，仅此而已。

首先需要安装 Package Control 因为 Sublime Text 的大部分插件都要借助这个工具来安装。翻译 Package Control 介绍的最简单的安装方法是通过 Sublime Text 控制台。可通过 control + ～ 快捷键或 View > Show Console 菜单访问控制台。 打开后，将适用于您的 Sublime Text 版本的 Python 代码粘贴到控制台中。

而我可能是找到了一个安装 Package Control 捷径：通过 Shift + Command + P 快捷键调出命令面板，键入 install Package Control 就安装成功了！

我使用的是 Sublime Text3 ，以安装以及删除 SCSS 语法高亮插件为例！

  • 1、通过 Preference > PackageControl 菜单打开命令面板
  • 2、通过 Shift + Command + P 快捷键打开命令面板

安装 SCSS 语法高亮插件

打开命令面板后，选择 Package Control: install Package ，然后输入 SCSS 可以看到插件列表，鼠标选择就开始安装了，也或者键盘上下键选择，然后按下 回车键开始安装。

删除 SCSS 语法高亮插件

打开命令面板后，选择 Package Control: install Package ，键盘上下键选择要删除的插件，然后按下 回车键，或者鼠标点击选中 就可以删除了。

喜欢的插件

  • Package Control 必备
  • Seti_UI 一个没有 Material 那么名气大的 Theme
  • Emmet 编码快捷
  • All Autocomplete 可补全 CSS 类名，官网描述：

    Extends the default autocomplete to find matches in all open files.
    By default Sublime only considers words found in the current file.

  • Auto​File​Name 自动完成文件名的输入，输入“/”即可看到相对于本项目文件夹的其他文件
  • Js​Format Javascript 代码格式化，在已压缩的JS文件中，右键选择jsFormat
  • CSScomb CSS 属性排序，让你的代码更漂亮
  • SASS 语法高亮
  • LESS 语法高亮
  • Stylus 语法高亮
  • Bracket​Highlighter 代码匹配，可匹配[], (), {}, “”, ”, 高亮标记，便于查看起始和结束标记
