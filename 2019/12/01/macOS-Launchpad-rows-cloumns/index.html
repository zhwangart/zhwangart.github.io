macOS 改变 Launchpad 图标的行列数

2019-12-01

macOS Launchpad（启动台） 中的 APP 图标默认 5 行 7 列显示，看起来图标太大，所以如果要改变图标大小，就得改变其显示的行列数。

改变行数量

defaults write com.apple.dock springboard-rows -int X

改变列数量

defaults write com.apple.dock springboard-columns -int X

将 X 改为想要设置的数字
提示：千万不要改太小了，建议 13.3 笔记本上，最佳是一页5行，每行9个（rows=5，columns=9）

使改变生效，重置 Dock

killall Dock

可以将所有指令放到一起，每一个句用 ; 分隔

恢复原样

defaults write com.apple.dock springboard-rows Default

defaults write com.apple.dock springboard-columns Default

killall Dock

重置 Dock 栏 APP

defaults delete com.apple.dock; killall Dock

如果Dock 栏上放满各类 App，却发现很多不是自己想要的App。可以通过终端重置 Dock 栏，回到系统默认状态。
