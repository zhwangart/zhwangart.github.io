精心挑选的 Photoshop 45 个快捷键

2019-02-22

每天都在使用 Photoshop 从 8.0 版本一直到现在的 CC 2019，由于不是介绍 Photoshop 版本的 Blog 所以不过多的贴图片了。参考 优设网的一篇译文 精心挑选了 Phtoshop 的 45 个快捷键，整理了一下方便学习。

由于现在 Mac 上一些修饰键符号已经没有了文字标注说明，所以首先标注说明一下 Mac 键盘上常用的几个修饰键符号。

符号 修饰键
⌘ Command（或 Cmd）
⇧ Shift
⌥ Option（或 Alt）
⌃ Control（或 Ctrl）
⌫ Delete（或 Backspace）
⇪ Caps Lock

在 Windows PC 专用键盘上，请用 Alt 键代替 Option 键，用 Windows 标志键代替 Command 键。

图层

功能 Mac OS Windows
填充图层 ⌘ + ⌫ (背景) or ⌥ + ⌫ (前景) Ctrl + Delete (背景) or Alt + Delete (前景)
平面化图层（盖印图层） ⌘ + ⌥ + ⇧ + E Ctrl + Alt + Shift + E
合并可见图层 ⌘ + ⇧ +E Ctrl + Shift + E
复制图层 ⌘ + J Ctrl + J
剪切图层 ⌘ + ⇧ + J Ctrl + Shift + J
上移图层至顶端 ⌘ + ⇧ + ] Ctrl + Shift + ]
下移至图层底端 ⌘ + ⇧ + [ Ctrl + Shift + [
上移动图层 ⌘ + ] Ctrl + ]
下移图层 ⌘ + [ Ctrl + [
复制多个图层 ⌘ + ⇧ + C Shift + Ctrl + C

选择

功能 Mac OS Windows
重新选择 ⌘ + ⇧ + D Ctrl + Shift + D
选择反向 ⌘ + ⇧ + I Ctrl + Shift + I
选择所有图层 ⌘ + ⌥ + A Ctrl + Alt + A
从选区中取消选择一部分 ⌥ + 拖移 Alt + 拖移
取消选择选区中未选中的区域 ⇧ + ⌥ + 拖移 Shift + Alt + 拖移
取消选区 ⌘ + D Ctrl + D
寻找变换点
自由变换时，图像若太大，无法看清变换点，可按Ctrl + 0		 ⌘ + T, ⌘ + 0 Ctrl + T, then Ctrl + 0
选择色彩通道 ⌘ + 3（红）,4（绿）,5（蓝） Ctrl + 3（红）,4（绿）,5（蓝）

画笔/填充

功能 Mac OS Windows
增加 / 减小画笔大小 ] / [ ] / [
增加 / 减小画笔硬度 } / { } / {
前一个 / 下一个笔刷 , / . , / .
第一个 / 最后一个笔刷 < / > < / >
填充
填充前景色或图案		 ⇧ + F5 Shift + F5
开关喷枪选项 ⇧ + ⌥ + P Shift + Alt + P

保存/关闭

功能 Mac OS Windows
保存，调出选项框 ⌘ + ⇧ + ⌥ + S Ctrl + Shift + Alt + S
关闭并自动打开 Adobe Bridge ⌘ + ⇧ + W Ctrl + Shift + W

图像

功能 Mac OS Windows
色阶 ⌘ + L Ctrl + L
自由变换 ⌘ + T Ctrl + T
曲线 ⌘ + M Ctrl + M
色彩平衡 ⌘ + B Ctrl + B
色相 / 饱和度 ⌘ + U Ctrl + U
去色
直接变为灰度图		 ⌘ + ⇧ + U Ctrl + Shift + U
内容识别比例
相对于自由变换，内容识别比例的效果是保持人物或者建筑等不进行特大幅度形变。		 ⌘ + ⇧ + ⌥ + C Ctrl + Shift + Alt + C
创建剪贴蒙版 ⌘ + ⌥ + G Ctrl + Alt + G
混合模式 ⇧ + 加号 or 减号 Shift + 加号 or 减号
黑白调整 ⌘ + ⇧ + ⌥ + B Shift + Ctrl + Alt + B

视图

功能 Mac OS Windows
实际像素 ⌘ + ⌥ + 0 Ctrl + Alt + 0 /Ctrl + 1
按屏幕大小缩放 ⌘ + 0 Ctrl + 0
放大 ⌘ + 加号 Ctrl + 加号
缩小 ⌘ + 减号 Ctrl + 减号

文本

功能 Mac OS Windows
增加/减小所选字符的磅数(2pts) ⌘ + ⇧ + > / < Ctrl + Shift + > / <
文本对齐 左/中/右 ⌘ + ⇧ + L / C / R Ctrl + Shift + L / C / R
对所选字符进行字距调整 ⌥ + 左右键 Alt + 左右键
