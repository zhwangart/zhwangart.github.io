Ocean 中使用 Valine

2020-02-22

Valine 是一款基于 LeanCloud 的快速、简洁且高效的无后端评论系统。现已集成到 Ocean 中，欢迎使用，或者一起设计 or 开发 Ocean ～

快速开始

在 Ocean 中使用 Valine 只需要在 ocean/_config.yml 中输入 LeanCloud 创建应用中的 appId 和 appKey 的值即可，请参照如下简单的配置：

获取 APP ID 和 APP Key

  1. 登录 LeanCloud 创建应用；
  2. 应用创建好以后，进入刚刚创建的应用，选择左下角的 设置 > 应用Key，然后就能看到你的 appId 和 appKey 了。

如果需要更详细的图文帮助，请参照： Valin 官方提供的文档

启用 Valine

  1. 启用 Valine：enable: false；
  2. 复制 appId 和 appKey 的值，粘贴到 ocean/_config.yml 对应的位置；
  3. 大功告成。

配置项

配置说明

配置项 说明
enable 是否启用，默认：false（关闭），true（启用）
el Valine 的初始化挂载器，无需更改
appId 从 LeanCloud 的应用中得到的 appId
appKey 从 LeanCloud 的应用中得到的APP Key
notify 评论回复邮件提醒，默认：false（关闭），true（启用）
verify 验证码，为避免过多的垃圾评论，所以默认 true（启用）
avatar Gravatar 头像展示方式，头像配置
pageSize 评论列表分页，每页条数
placeholder 评论框占位提示符

如需要了解更多配置项，请参照：Valine 配置项。
