可适用于企业的 Hexo 主题 Acorn

2022-03-04

设计了一“颗”可适用于中小型企业门户或工作室等展示类网站的 Hexo 主题： Acorn ，免费且开源，个人或企业均可免费使用，但请不要用于商业使用。

  • GitHub：https://github.com/zhwangart/hexo-theme-acorn
  • 样例数据：https://github.com/zhwangart/acorn-sample-data
  • 预览：https://acorn.imaging.xin/
  • 文档：https://acorn.imaging.xin/docs/

为什么叫 Acorn 这个名字？
没什么深意，致敬《ICE AGE》这部很喜欢的电影吧～

所以我们还设计了一个很符合它的 404 页面：

Hexo 主题列表目前已经有 350+ 个优秀开发者们贡献的主题，这些主题多用于兴趣爱好或学习博客，更有无数人使用 Hexo 建立了自己的站点/博客。那么为什么我们设计了一“颗”适用于企业站点的主题？

首先 Hexo 有如下优点：

  • 免费 - GitHub Pages 一键部署
  • 快速 - Node.js 运行环境，上百个页面在几秒内瞬间完成渲染
  • 简洁 - MarkDown 简单直观
  • 轻量 - 全部静态文件

其次通常小型的企业站点文章数量远不及大 V 博主，企业更注重形象展示、内容表现，所以我们转移一下设计重点：个人秀转企业秀。

最后，学习成本极低，抓个文职妹子就把公司网站建立起来了，我们不止是写了详细的 使用文档，并且还准备一份完整的 样例数据,将 source 文件夹中的全部内容 Copy 至你站点对应的 source 文件夹中，刷新页面后即可看到使用该样例数据渲染的效果。
